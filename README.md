## Tentang Diginest Agency

**Diginest Agency** adalah partner terpercaya dalam pengembangan website profesional. Kami hadir untuk membantu bisnis Anda tampil profesional di dunia digital dengan website yang modern, responsif, dan sesuai kebutuhan.

**Website:** [https://diginest.id](https://diginest.id)

## Layanan Kami

- **Landing Page** - Website satu halaman untuk promosi produk/jasa
- **Company Profile** - Website profil perusahaan yang profesional
- **Toko Online** - E-commerce dengan sistem pembayaran terintegrasi
- **Web Custom** - Aplikasi web sesuai kebutuhan bisnis
- **Maintenance** - Layanan perawatan dan update website

## Tech Stack

- React 18
- TypeScript
- Styled Components
- Ant Design
- i18next

## Instalasi

```bash
# Clone repository
git clone https://github.com/SeptianSamdani/diginest-landing-page.git

# Masuk ke direktori
cd diginest-landing-page

# Install dependencies
npm install

# Jalankan development server
npm start
```

Website akan berjalan di `http://localhost:3000`

## Build Production

```bash
npm run build
```
